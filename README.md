# Student Success Predictor: My First ML Model

## 📌 Project Overview
This project is an introductory study on **Linear Regression**. I built a machine learning model to predict a student's test score based on the number of hours they studied.

The goal was to move beyond simple "if-else" logic and let Python discover the relationship between study habits and academic results.

## 🛠️ Tech Stack
* **Language:** Python 3
* **Libraries:** * `NumPy`: For numerical data processing.
    * `Scikit-Learn`: To build and train the Linear Regression model.
    * `Matplotlib`: To visualize the results and the "Line of Best Fit."

## 📊 The "Line of Best Fit"
The model calculates a linear equation ($y = mx + b$) to represent the trend. Even when data is scattered, the model finds the path that minimizes error for the entire group.



## 🧪 Key Learning: The Impact of Outliers
During this project, I experimented with adding "Outliers" (e.g., a student who studied very little but scored very high). I observed how a single data point can "tilt" the model's logic, highlighting the importance of **Data Cleaning** before training a model.
